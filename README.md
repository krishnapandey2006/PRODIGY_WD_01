# PRODIGY_WD_01 — Responsive Landing Page
**Task 1** for Prodigy InfoTech Web Development Internship (December 2025)

## Live Demo
https://krishnapandey2006.github.io/PRODIGY_WD_01/

## Features
- Responsive hero + grid layout
- Sticky navbar that changes on scroll
- Mobile hamburger menu
- Smooth scroll & active link highlight
- Accessible semantic markup

## Tech
- HTML5, CSS3, Vanilla JavaScript

## How to run locally
1. Clone: `git clone https://github.com/krishnapandey2006/PRODIGY_WD_01.git`
2. `cd PRODIGY_WD_01`
3. Open `index.html` in browser (or run `npx http-server` for a local server)
